# Train Ticket Booking System (C++)
# Description

* This is a simple Train Ticket Booking System written in C++. It uses Object-Oriented Programming (OOP) to handle basic ticket operations like booking, cancellation, and displaying ticket details.

# Features
* Book a train ticket
* Cancel a booked ticket
* Display ticket details
* Menu-driven user interface
# How It Works
* The program uses a Train class to store passenger information such as name, age, and seat number.
* A booked flag is used to check whether a ticket is already booked or not.
* The user interacts with the system using a menu in the main() function.
# Operations
* Book Ticket – Enter passenger details and reserve a seat.
* Cancel Ticket – Cancel the current booking.
* Display Ticket – Show booked ticket information.
* Exit – Close the program.
# Concepts Used
* Classes and Objects
* Encapsulation
* Member Functions
* Conditional Statements
* Loops
* User Input/Output
